About Matric Verification

This system uses on-chain cryptographic identity binding. Each student's matric number is hashed using keccak256 and permanently stored on the blockchain.


This ensures that:

  • One matric number = one wallet = one vote
  • The same matric number cannot register twice
  • Raw matric numbers are never stored publicly
  • Verification is done purely on-chain

About Voter Certificates

Every voter automatically receives a non-transferable NFT after casting their vote. This certificate:

  • Cannot be transferred or sold
  • Is permanently bound to the voter's wallet
  • Contains the election title and timestamp
  • Serves as verifiable proof of participation
  • Can be viewed in MetaMask under NFTs
